## ⚡ 30-Second Setup

1. **Get your API key** from chosen provider (links below)
2. **Add to `.env.local`:**
```
ANTHROPIC_API_KEY=sk-ant-xxxxx
NEXT_PUBLIC_AI_PROVIDER=claude
```
3. **Restart:** `npm run dev`
4. **Test:** Try auto-categorizing an expense
